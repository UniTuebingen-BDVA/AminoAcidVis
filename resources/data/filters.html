Filters

Filters are a powerful tool to manipulate the data you see, to hide certain items or change its appearance.
This is done by submitting C++ inspired expressions, for example AtomElement == H (Atom element is equal H) will hide all hydrogen (H) atoms from the timeline and render views.
There are two types of expressions distinct by there return type:

  • Boolean - Expressions that return true or false. They will hide a item or items from the data if true.
    (Example: ResidueName == CYS)
  • Numerical - Expressions that return a float or integer. They will set the heatmap with the returned value.
    (Example: 42 or ResidueLayer(frame) - ResidueLayer(Tracker(0)))

All other expressions will be ignored and will not have any effect.

Comparison

Comparison operator always return a boolean value.

Syntax Description
a == b True, if a equal b.
a != b True, if a not equal b.
a < b True, if a is bigger then b.
a > b True, if a is smaller then b.
a <= b True, if a is bigger then or equal b.
a >= b True, if a is smaller then or equal b.

Logical

Comparison operator always return a boolean value.

Syntax Description
a & b True, if a and b are true.
a | b True, if a or b is true.
